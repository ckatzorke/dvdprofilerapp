DVDProfiler CouchApp - Thumbnail Gallery


Data is provided by DVDProfiler from invelos, an awesome program to manage your collection.
Following is a sample thumbnail gallery.
  • It uses carouFredSel as a jQuery carousel plugin
  • List is generated by a view, that emits all docs with the purchase date as key (decending)
  • Thumbnails are loaded lazily, inspired by jQuery lazy load, the images are loaded after they entered the viewable area (better would be, before they enter the viewport)
  • All items are loaded, since dynamically inserting items behaves really strange
  • not yet responsive :-(
